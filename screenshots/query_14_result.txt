# Q14. 전체 도서 평균 가격보다 비싼 도서 목록 조회 (비교 단일행 서브쿼리)
# Query:
SELECT title, price 
FROM book 
WHERE price > (SELECT AVG(price) FROM book);

# Result:
+------------------+-------+
| title            | price |
+------------------+-------+
| 이것이 MySQL이다 | 30000 |
| 맛있는 파이썬    | 25000 |
| 사피엔스         | 22000 |
| 코스모스         | 20000 |
+------------------+-------+
